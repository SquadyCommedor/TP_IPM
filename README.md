# O Meu Guia do Cabeleireiro - v2.0

Aplicação para crianças com PEA (Perturbação do Espetro do Autismo) que auxilia na preparação e acompanhamento durante visitas ao cabeleireiro, com monitorização de stress via pulseira BITalino.

## Novidades v2.0

- **Layout Responsivo**: Funciona perfeitamente em telemóvel e computador
- **Design Moderno**: Interface colorida e apelativa para crianças
- **Ícones Lucide**: Biblioteca de ícones consistente em toda a app
- **Imagens Reais**: Cenas com fotografias reais de salões de cabeleireiro
- **Animações Framer Motion**: Transições suaves e interativas
- **Navegação Adaptativa**: Bottom nav no mobile, sidebar no desktop
- **Exercício de Respiração**: Modal animado para acalmar
- **Dashboard Pais Melhorado**: Estatísticas visuais e filtros

## Instalação

```bash
npm install
```

## Configuração

1. Copia `.env.example` para `.env`
2. Preenche com os dados do teu projeto Supabase

## Desenvolvimento

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Stack

- React 19 + TypeScript + Vite
- Tailwind CSS v4
- Framer Motion
- Lucide React (ícones)
- Supabase (Auth + DB)
- React Query
- React Hot Toast
